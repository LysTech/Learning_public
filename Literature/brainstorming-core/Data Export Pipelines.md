## Key points
- Automated pipeline for converting raw fNIRS channel data to HbO/HbR hemoglobin concentrations
- Supports export to multiple data formats for downstream analysis
- Accessed through the Pipelines tab in the Kernel Portal interface
- Enables standardized preprocessing from raw acquisition to reconstructed neuroimaging data
