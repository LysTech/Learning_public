## Summary
This work demonstrates robotic hand control using a brain-computer interface where EEG signals provide direct feedback integrated into the reinforcement learning reward function, enabling neural feedback-guided learning.

## Key points
- EEG-based BCI for robotic hand manipulation using reinforcement learning
- Incorporates neural feedback (EEG signals) directly into the RL reward signal
- Demonstrates improved control performance when EEG feedback is included in learning
- Yellow line results show benefits of neural-guided reward shaping over standard RL
