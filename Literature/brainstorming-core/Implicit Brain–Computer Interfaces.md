## Summary
This work presents the concept of implicit brain-computer interfaces (iBCIs), where users do not deliberately control the system but rather the BCI infers cognitive state from brain activity to enhance human-machine interaction. Section 4 demonstrates practical applications including cursor control using implicit BCI principles.

## Key points
- Introduces implicit BCIs that infer user cognitive state without explicit voluntary control
- Contrasts with traditional explicit BCIs where users deliberately modulate brain activity
- Demonstrates practical cursor control application using implicit BCI in Section 4
- Emphasizes potential for more natural and less cognitively demanding human-computer interaction
