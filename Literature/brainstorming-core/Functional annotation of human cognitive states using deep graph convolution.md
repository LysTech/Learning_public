## Summary
This paper presents a deep graph convolutional network approach for functional annotation of human cognitive states from fMRI data. The method leverages brain connectivity patterns to classify and characterize different cognitive states, demonstrating improved performance over traditional approaches in neuroimaging analysis.

## Key points
- Uses graph convolutional networks to leverage brain connectivity information for cognitive state classification
- Provides functional annotation of cognitive states from fMRI data
- Demonstrates improved classification performance compared to standard neuroimaging methods
- Combines deep learning with neuroscience-informed graph-based representations of brain activity
