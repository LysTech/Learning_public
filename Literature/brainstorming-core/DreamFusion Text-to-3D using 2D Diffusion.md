## Summary
DreamFusion: Text-to-3D using 2D Diffusion, 2022.

## Key points
- Text-to-3D generation using only 2D diffusion models as guidance, avoiding the need for paired text-3D training data
- Introduces score distillation sampling (SDS) to distill knowledge from 2D diffusion models into 3D representations
- Generates diverse, detailed 3D models from arbitrary text prompts with coherent geometry and appearance
- Outperforms prior text-to-3D methods and enables practical applications in creative 3D content generation
