## Summary
This paper explores methods for scaling instructable agents through reinforcement learning with prior knowledge. The work addresses how to effectively incorporate high-level instructions and priors to improve agent learning and generalization across diverse tasks.

## Key points
- Proposes methods to scale instructable agents using reinforcement learning
- Investigates the use of abandon-grade priors to improve learning efficiency
- Focuses on agent generalization and instruction-following capabilities
- Evaluates scalability across multiple tasks and environments
