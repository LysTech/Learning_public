## Summary
When not using reasoning, repeating the input prompt improves performance for popular models (Gemini, GPT, Claude, and Deepseek) without increasing the number of generated tokens or latency.

## Key points
- Simple prompt repetition strategy improves non-reasoning LLM performance across multiple architectures
- Method adds no latency or token overhead compared to standard inference
- Evaluated on Gemini, GPT, Claude, and Deepseek models
- Demonstrates practical improvement mechanism without computational cost
