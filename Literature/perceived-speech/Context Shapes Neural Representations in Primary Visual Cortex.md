## Summary
A review examining how contextual information shapes neural representations in primary visual cortex, discussing mechanisms by which surrounding stimulus context modulates neural responses and alters the encoding of visual features.

## Key points
- Context dramatically modulates neural responses in V1 beyond simple feedforward mechanisms
- Surround suppression and facilitation are key contextual effects that reshape visual representations
- Multiple neural circuits and feedback pathways implement context-dependent processing
- Contextual modulation improves population coding efficiency and behavioral discrimination
