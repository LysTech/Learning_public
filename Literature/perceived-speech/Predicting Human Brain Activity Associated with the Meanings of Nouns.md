## Summary
This landmark study demonstrates that fMRI brain activation patterns evoked by viewing concrete nouns can be predicted using semantic features derived from large text corpora. The work shows that abstract semantic properties of words have consistent neural representations across subjects.

## Key points
- Predicts fMRI activation patterns for 60 concrete nouns using semantic feature vectors
- Semantic features extracted from text corpora capture neural representations of word meanings
- Cross-subject generalization shows consistency in how brains encode semantic information
- Foundational work linking computational semantics to neuroscience
- Includes publicly available dataset widely used in subsequent studies
