## Summary
Voyager is an embodied agent in Minecraft that uses large language models to autonomously explore, set goals, and solve complex tasks without human intervention. The system combines LLMs with skill learning and curriculum generation to achieve open-ended exploration and task solving.

## Key points
- First open-ended embodied agent powered by LLMs that operates autonomously in complex 3D environments
- Uses LLMs for goal generation, planning, and reflection on agent experiences
- Learns and accumulates reusable skills across multiple tasks without human guidance
- Achieves complex multi-stage objectives in Minecraft including mining rare materials and building structures
