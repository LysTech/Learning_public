## Summary
This paper proposes ChannelNets, a neural network architecture that uses channel-wise convolutions to handle variable input dimensions and reduce computational complexity while maintaining model efficiency.

## Key points
- Introduces channel-wise convolution operations for handling variable channel dimensions
- Reduces model parameters and computational cost compared to standard convolutions
- Achieves competitive performance on image classification with improved efficiency
- Applicable to networks with varying input channel requirements
