## Summary
This work presents methods for visual semantic decoding and functional near-infrared spectroscopy (fNIRS) reconstruction using subject-specific anatomical information, addressing the limitations of template-based approaches.

## Key points
- Proposes fNIRS reconstruction using subject-specific brain anatomy rather than template-based approaches
- Demonstrates visual semantic decoding capabilities on reconstructed fNIRS data
- Addresses mismatch between individual anatomy and group-averaged templates like Colin27
- Relevant for real-time applications where live reconstruction is required
