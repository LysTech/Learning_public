## Summary
This work analyzes resting state fMRI data using deconvolution techniques to reveal network connectivity patterns not apparent in the raw BOLD signal, relevant for studying spontaneous brain activity without a stimulus protocol.

## Key points
- Applies deconvolution analysis to resting state fMRI data
- Reveals network connectivity features hidden in the convolved BOLD signal
- Addresses the challenge of analyzing resting state data without a stimulus protocol
- Demonstrates applicability of spectral techniques to resting state neuroimaging
