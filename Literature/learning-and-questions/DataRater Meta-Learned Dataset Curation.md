## Summary
DataRater proposes a meta-learned approach for automatic dataset curation that learns to identify and select high-quality training examples. The method uses meta-learning to optimize data selection strategies across diverse tasks, improving model performance while reducing computational overhead.

## Key points
- Meta-learned framework for automatic dataset curation that generalizes across tasks
- Learns to identify high-quality training examples without manual annotation
- Reduces computational cost while maintaining or improving model performance
- Demonstrates effectiveness on multiple domains and dataset scales
