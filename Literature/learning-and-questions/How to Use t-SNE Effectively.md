## Summary
Although extremely useful for visualizing high-dimensional data, t-SNE plots can sometimes be mysterious or misleading.

## Key points
- t-SNE is powerful for visualization but can be easily misinterpreted without understanding its behavior
- Key parameters like perplexity and learning rate significantly affect output and require careful tuning
- Cluster size and spacing in t-SNE plots do not reflect actual data properties and can be misleading
- Proper validation and comparison with other dimensionality reduction methods is essential
- Understanding t-SNE's local neighborhood preservation helps correct common misreadings of plots
