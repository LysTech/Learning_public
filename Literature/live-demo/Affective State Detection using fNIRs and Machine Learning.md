## Summary
This paper presents a machine learning approach for detecting affective states using functional near-infrared spectroscopy (fNIRS) neuroimaging data. The work leverages fNIRS signals to classify emotional or mental states with various ML algorithms.

## Key points
- Applies machine learning to fNIRS-based brain imaging for affective state classification
- Utilizes hemodynamic signals from functional near-infrared spectroscopy as input features
- Demonstrates feasibility of non-invasive optical neuroimaging for emotion/mental state detection
- Relevant for brain-computer interfaces and affective computing applications
