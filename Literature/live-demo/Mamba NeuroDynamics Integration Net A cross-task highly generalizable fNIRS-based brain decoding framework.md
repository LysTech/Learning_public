## Summary
This paper presents Mamba NeuroDynamics Integration Net, a novel fNIRS-based brain decoding framework that leverages state-space models for cross-task generalization. The approach achieves high accuracy in decoding brain activity across multiple cognitive tasks, demonstrating superior generalization capabilities compared to existing methods.

## Key points
- Introduces Mamba architecture for fNIRS-based brain decoding with improved cross-task generalization
- Demonstrates state-of-the-art performance across multiple cognitive task datasets
- Achieves high generalization capability when tested on unseen tasks and subjects
- Combines neuroscience domain knowledge with modern deep learning architectures for robust brain signal interpretation
