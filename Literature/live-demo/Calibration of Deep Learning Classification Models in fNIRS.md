## Summary
This paper addresses the calibration of deep learning classification models applied to functional near-infrared spectroscopy (fNIRS) data, examining how well model confidence scores align with actual prediction accuracy. The work emphasizes the importance of calibration for reliable uncertainty estimation in brain-computer interface and neuroscience applications.

## Key points
- Examines calibration of deep learning classifiers for fNIRS brain activity classification
- Demonstrates miscalibration in standard neural network models on fNIRS data
- Proposes calibration techniques to align model confidence with actual accuracy
- Relevant for reliable uncertainty quantification in BCI and neuroscience applications
