## Summary
This paper introduces sparsemax, a sparse alternative to softmax that enables truly sparse attention weights and multi-label classification. Unlike softmax which always produces dense probability distributions, sparsemax can output exact zeros, making it more interpretable and suitable for problems requiring sparse selections.

## Key points
- Proposes sparsemax as a sparse alternative to softmax that can assign exactly zero probability to irrelevant elements
- Demonstrates advantages for attention mechanisms by enabling more interpretable, sparse attention patterns
- Extends framework to multi-label classification where multiple labels can be selected simultaneously
- Provides efficient algorithms for computing sparsemax in both forward and backward passes
- Shows empirical improvements over softmax in multi-label classification tasks
