## Summary
Deception is a human behavior that many people experience in daily life. It involves complex neuronal activities in addition to several physiological changes...

## Key points
- Combines fNIRS brain imaging with polygraph physiological measurements for enhanced lie detection
- Demonstrates single-trial detection capability, enabling real-time deception identification
- Integrates hemodynamic (prefrontal cortex) and autonomic nervous system responses
- Achieves improved classification performance through multimodal data fusion
