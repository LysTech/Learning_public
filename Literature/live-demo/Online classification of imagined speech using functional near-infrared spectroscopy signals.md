## Summary
This paper presents an online classification system for imagined speech using functional near-infrared spectroscopy (fNIRS) signals. The work demonstrates real-time decoding of covert speech from brain hemodynamic activity, achieving practical performance for brain-computer interface applications.

## Key points
- Real-time online classification of imagined speech from fNIRS brain signals
- Demonstrates practical brain-computer interface for speech decoding without overt vocalization
- Achieves clinically relevant performance metrics for assistive communication applications
- Uses hemodynamic responses from prefrontal cortex regions associated with speech production
